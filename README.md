# Banking System

A console-based banking application built in Java for a Structured Programming course at Abu Dhabi University.

## Features
Customer
- Create an account with a unique auto-generated 6-digit account number
- Secure login with a 3-attempt limit
- Deposit, withdraw, and transfer money between accounts
- View balance and delete account

Admin
- Separate login with admin credentials
- View all accounts with names and balances
- Delete any account

## Password Requirements
- Minimum 8 characters
- Must contain at least one special character: `!@#$%`

## Technologies
- Java
- NetBeans IDE

## Notes
All data is stored in-memory using arrays and resets on program exit, no database used.
